Lab 3: Glyph-based Visualizations Lab

The focus of this lab was on developing skills in ArcGIS and Python for data visualization. The objectives included using the ArcGIS Charts renderer, performing basic data matching, using Python to read GIS attribute tables, and automating workflows. I prepared the data by creating datasets, and created glyph-based maps with charts and glyphs.

Lab 4: Kepler Lab

This lab provided an overview of using Kepler.gl, an open source data visualization tool, to visualize crime data in Raleigh, North Carolina from the city's open data portal. In this lab, we explored customzing visualizations using marks, channels, filters and radius. We also learned how to view data from a 3D perspective. Overall, the tutorial offered a practical introduction to using Kepler.gl for spatial-temporal data visualization.
